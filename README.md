<h2>Persyaratan Awal:</h2>

<ul>
  <li>Pastikan Anda memiliki Python terinstal di sistem Anda. Jika belum, Anda dapat mengunduh dan menginstal Python dari <a href="https://www.python.org/downloads/">python.org</a>.</li>
  <li>Pastikan Anda memiliki koneksi internet untuk mengakses API NewsAPI.</li>
  <li>Pastikan Anda telah mendaftar di <a href="https://newsapi.org/">NewsAPI</a> dan mendapatkan API key. Ganti <code>apiKey="f38b26e86acb46c29058439829075408"</code> dengan kunci API Anda.</li>
</ul>

<h2>Langkah-langkah Penggunaan:</h2>

<ol>
  <li>Unduh atau salin kode skrip Python ke sistem Anda.</li>
  <li>git clone https://github.com/Yoga913/Pembaca-Berita-Utama/<li>
  <li>Pastikan Anda telah menginstal pustaka yang diperlukan, yaitu <code>requests</code> dan <code>pyttsx3</code>. Anda dapat menginstalnya menggunakan pip dengan menjalankan perintah berikut di terminal atau cmd:
    <pre><code>pip install requests pyttsx3</code></pre>
  </li>
  <li>Ubah <code>url</code> dengan API key Anda jika diperlukan.</li>
  <li>Jalankan skrip Python dengan menjalankan perintah berikut di terminal atau cmd:
    <pre><code>python news.py</code></pre>
  </li>
</ol>
